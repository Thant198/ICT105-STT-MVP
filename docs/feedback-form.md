## Purpose

This questionnaire collects both quantitative and qualitative feedback from users after they complete the MVP usability testing script. The questions are written in a neutral and non-leading way to identify confusion points, evaluate screen layout usefulness, and understand users' true expectations.

## Section G – Open Feedback

### 32. What was the biggest difficulty you experienced?

The biggest difficulty was understanding whether some actions had been completed successfully because confirmation messages were not always noticeable.

### 33. Is there anything you particularly liked about the system?

I liked the simple dashboard, the Assignment Tracker, and the Pinned Links Dashboard because they made important student information easy to access.

### 34. What improvements would make the system more useful?

Adding notifications, deadline reminders, assignment priorities, search, filtering, and a calendar view would make the system more useful.

### 35. What did you like most about the system?

I liked having assignments and important links in one centralized dashboard.

### 36. What did you like least about the system?

Some buttons and status changes were not clear enough, and the system needed stronger visual feedback after user actions.

### 37. What additional features would you like to see?

I would like to see:

- Calendar integration
- Deadline notifications
- Assignment priority levels
- Search and filter controls
- Team collaboration
- File attachments
- Progress tracking
- Dark mode

### 38. Additional comments or suggestions

The MVP is useful and easy to learn. The interface would be more effective with clearer buttons, stronger confirmation messages, improved status indicators, and better use of empty dashboard space.

## Evaluation Summary

The participant successfully completed all required tasks within two minutes. The Assignment Tracker received a usefulness rating of **4/5**, and the Pinned Links Dashboard received **5/5**. The participant found the overall layout clear but identified confusion around save confirmation, status changes, and editing controls.

The main requested improvements were:

- Clearer success notifications
- Calendar and reminder features
- Better assignment sorting and priorities
- Search and filtering
- More detailed status information
