# Product Concept - Campus Life Hub

### 1. Target User
Mainly university/ICT students who participate in group projects and student club organizers.

### 2. Core Problem
Platform Fragmentation. Critical project updates, assignment deadlines, and event information are heavily scattered across Facebook, Messenger, LINE, Email, and LMS, causing students to miss deadlines and lose important links.

### 3. Proposed System
"Campus Life Hub" – A centralized web-based dashboard and student project management system that consolidates announcements, unified milestone calendars, and group project repositories into a single platform.

### 4. Value Proposition
Eliminates communication noise and coordinates scattered university workflows by providing one single source of truth for all project and campus deadlines.

### 5. MVP Scope (In-Scope)
- Centralized Announcement Dashboard (F01)
- Unified Deadline Calendar with visual status tags (F02)
- Project-Specific Shared Links Repository for group collaboration (F03)

### 6. Out-of-Scope (Excluded)
- Automatic mobile push notifications (F04)
- AI-Powered Smart Syllabus Chatbot (F05)
