## Reflection

### Which feature creates the strongest user value?

**Manage Project Deadlines (F-02)** creates the strongest user value because it helps students avoid missing important deadlines, which was one of the most common problems identified during Lab 03 customer discovery.

### Which feature creates the strongest business or operational value?

**Admin Record Management (F-06)** creates the strongest operational value because it allows administrators to maintain accurate, up-to-date, and reliable information throughout the system.

### Which required feature is still weak or unclear?

**Submit Announcements / Projects / Events (F-04)** is still relatively weak because user permissions and the approval workflow need to be defined more clearly before implementation.
