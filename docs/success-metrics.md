# MVP Success Metrics

**Project:** Campus Life – Student Project Management System  
**Deliverable:** `/docs/success-metrics.md`

## Objective

This document defines the measurable success criteria for evaluating the MVP (Minimum Viable Product) during usability testing. The experiment will determine whether the Assignment Tracker and Pinned Links Dashboard effectively help students organize academic work.

---

## Success Metrics

| Metric | Target Benchmark | Measurement Method |
|--------|------------------|--------------------|
| Task Completion Rate | **≥ 70%** | Percentage of participants who successfully complete all assigned tasks without assistance |
| Average Time to Complete Tasks | **≤ 2 minutes** | Average time required to complete each core task |
| Assignment Tracker Usefulness | **≥ 4.0 / 5.0** | Post-test user survey (Likert Scale) |
| Pinned Links Dashboard Usefulness | **≥ 4.0 / 5.0** | Post-test user survey (Likert Scale) |
| Ease of Use | **≥ 4.0 / 5.0** | User experience questionnaire |
| Overall User Satisfaction | **≥ 80%** | Percentage of satisfied or very satisfied participants |
| Error Rate | **< 10%** | Percentage of failed actions or user mistakes |
| Feature Adoption | **≥ 80%** | Participants who successfully use both core features |

---

## Core User Tasks

Participants will complete the following tasks during testing:

1. Create a new assignment.
2. Edit assignment details.
3. Set a due date.
4. Mark an assignment as completed.
5. Add a pinned reference link.
6. Open a pinned link from the dashboard.

---

## Success Criteria

The MVP will be considered successful if all of the following conditions are met:

- ✅ Task completion rate is **70% or higher**
- ✅ Average task completion time is **2 minutes or less**
- ✅ Assignment Tracker usefulness rating is **4.0/5.0 or higher**
- ✅ Pinned Links Dashboard usefulness rating is **4.0/5.0 or higher**
- ✅ Overall user satisfaction reaches **80% or higher**
- ✅ Error rate remains **below 10%**
- ✅ At least **80%** of participants use both primary features successfully

---

## Data Collection

The following data will be collected during the MVP experiment:

- Number of successful task completions
- Time required for each task
- Number of user errors
- User satisfaction survey responses
- Assignment Tracker usefulness rating
- Pinned Links Dashboard usefulness rating
- User feedback and improvement suggestions

---

## Conclusion

Meeting these benchmarks will demonstrate that the MVP provides a usable, efficient, and valuable solution for helping university students manage assignments and access important project resources through a centralized dashboard.
