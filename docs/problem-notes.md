# Problem Notes

## Project Title
Campus Life - Student Project Management System

## Target Users
- University Students
- Club Leaders
- Event Organizers
- Lecturers

## Common Problems

### Problem 1
Students often miss important announcements.

### Problem 2
Students have difficulty tracking project deadlines.

### Problem 3
Students cannot easily find campus events and activities.

### Problem 4
Communication between project members is inconsistent.

## Current Alternatives

- Facebook Groups
- LINE Groups
- Messenger Groups
- Email
- Word of Mouth

## Consequences

- Missed deadlines
- Poor project coordination
- Low event participation
- Communication confusion
