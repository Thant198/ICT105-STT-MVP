# Campus Life Secure Prototype — Issue #64

This package contains a complete static HTML/CSS/JavaScript implementation for **ICT105 Lab 09 Issue #64: Implement Prototype Privacy and Security Mitigations in Code**.

## Files

```text
prototype/
├── index.html
├── css/
│   └── styles.css
└── js/
    └── app.js

docs/
└── ISSUE-64-IMPLEMENTATION.md
```

## Run

Open `prototype/index.html` in a browser. No installation is required.

## Add to GitHub

Copy the contents of the `prototype` folder into the repository's existing `prototype/` folder. Add `docs/ISSUE-64-IMPLEMENTATION.md` as implementation evidence, then commit using:

```text
feat: implement Lab 09 privacy and security mitigations (#64)
```

## Important Limitation

This is a static educational prototype. The role selector demonstrates authorization behavior in the interface, but real production authorization must be enforced by a secure server.
