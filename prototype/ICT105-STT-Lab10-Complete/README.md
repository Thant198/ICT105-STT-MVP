# Campus Life – Student Project Management System

ICT105-STT-MVP helps university students manage assignments, team projects, deadlines, and frequently used campus links.

## Lab 10 – MVP Implementation Sprint 1

### Run
Open `prototype/index.html` in a browser. No installation or backend is required.

### Implemented
- Responsive single-page navigation
- Assignment CRUD basics, JSON sample data, localStorage
- Search and status filter
- Dashboard totals and due-soon logic
- Team project progress and pinned links
- Admin status update
- Form validation and confirmation messages
- Student-ID masking and safe output handling

### GitHub Issues Covered
