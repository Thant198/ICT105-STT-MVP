# Data Directory

## Purpose

This folder stores all datasets used for the Campus Life – Student Project Management System (MVP) validation and usability testing.

These files are used to evaluate the prototype, calculate usability metrics, and support analytics reporting.

---

## Files

| File | Description |
|------|-------------|
| customer_segments_value.csv | Customer segmentation and value scoring data. |
| experiment-results.csv | Usability experiment results collected during MVP testing. |
| lab08_data_dictionary.csv | Data dictionary describing each dataset field. |
| lab08_validation_results.csv | Validation results collected from usability testing participants. |
| lost-found-sample-records.csv | Sample records used for testing data management features. |
| opportunity-scoring.xlsx | Opportunity scoring matrix for prioritizing product ideas. |
| raw-responses.xlsx | Raw participant responses collected during usability testing. |

---

## Core Metrics

The following metrics are calculated from the usability testing data.

### Success Rate

Measures the percentage of participants who completed the assigned tasks successfully.

**Formula**

```
Success Rate = (Completed Tasks ÷ Total Participants) × 100
```

Target: **70% or higher**

---

### Feedback Score

Measures the average participant satisfaction using a 5-point Likert scale.

**Formula**

```
Feedback Score = Total Feedback Score ÷ Total Participants
```

Target: **4.0 / 5.0 or higher**

---

### Interest Level

Measures the average participant interest in using the Campus Life MVP.

**Formula**

```
Interest Level = Total Interest Score ÷ Total Participants
```

Target: **4.0 / 5.0 or higher**
